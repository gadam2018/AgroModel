          AgroModel Agricultural Data Management System Version1.0
                           
               *********  Installation Notes  **********

1) The first time you run setup, is required to restart your system.
2) Then, once you restart, you run setup again.
   (Preferred installation dir: C:\AgroModel) 
   At the end of the setup, the following files will try to self register, but will fail.
       	DAXCTLE.OCX
	DATAFORM.OCX
	APPWIZ.OCX
   Choose Ignore.
